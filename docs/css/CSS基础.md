## 引入方式

### 行内样式

```html
<p style="color:red"></p>
```

### 内嵌样式

```html
<head>
  <style>
    p {
      color:red;
      }
  </style>
</head>
```

### 外联样式

```html
通过link标签引入
<link src="xxx"></link>
```
